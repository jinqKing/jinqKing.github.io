1. \(\LaTeX\)

想当初我也感到好看，正是今天我去找，发现这是十分优雅的，一个指令就让它完成了格式。然而这份优雅与我并不相干，我只是指令的记忆操作工，其中并没有几乎没有任何我可以着手改变的部分。我也不理解其中深刻的思想1

不过从一开始，我也就广纳多见，将它袪魅。我看到贴子说，有的人认为会 LaTeX 就，最开始应当是在知乎上看到

2. Emacs

现在慢慢感到它是很不方便，为了一些功能的实现我需要了解太多的内容这些付出，在无人指导的情况下我感到是不可接受的。主要讲来我使用它最大动力就是 org 使用哦 org 的好处是可以一键导出 pdf html，后者进一步构成我做博客的基础。比 markdown 的优势在于有 Emacs 这个编辑器的加持，部分操作更加便捷，尤其关于 latex 公式，我插入了太多快捷方式漫长沙离不开它们了。

但它教给了我很多，让我感到更接近底层地感到软件的运行，输入交互这一切都是各种数据的传输处理，敲击的键盘

我常常想，应当写一个当下的新时代的 Emacs 了，还用 lisp,不过应当更加通用一些为好。

我当然想不过目前要补习的内容太多。

不过也正是在这里，我可一看到这样的话：
